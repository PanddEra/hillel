Усі компоненти у цьому завданні повинні бути функціональними.

Компонент отримує дані через пропс — дані лежать у наданому масиві (див. “дані лежать тут”).

Необхідно прийняти цей пропс у компоненті та відрендерити отримані дані відповідно до заданої HTML-структури.

Підключіть компонент Definitions у App.js і відрендерте його.

Не забувайте про пропс key при рендері списків елементів.
